BeCyIconGrabber Portable Help

Grab icons easily using this tool!

BeCyIconGrabber Portable is a tool which allows you to extract icons from almost every file. It's packaged with a PortableApps.com launcher as a portable app, so you can use it from a cloud folder, portable drive, or local folder without needing to install it on each PC.
